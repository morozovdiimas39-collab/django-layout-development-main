# Auto-Blog — автопубликация статей

Функция генерирует статью через Gemini и добавляет в блог. Запускается по cron 5 раз в день.

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `GEMINI_API_KEY` | Ключ Gemini API |
| `DATABASE_URL` | Подключение к PostgreSQL |
| `SITE_URL` | (опц.) URL сайта для картинок, по умолчанию основной домен |
| `DB_SCHEMA` | (опц.) Схема БД. По умолчанию `t_p90119217_django_layout_develo` (из миграций) |
| `HTTPS_PROXY` | (опц.) HTTPS-прокси для запросов к Gemini, например `https://user:pass@host:443` |

## Cron в Yandex Cloud

Создай триггер типа «Таймер» с расписанием:

```
0 8,12,16,20,0 * * *
```

(8:00, 12:00, 16:00, 20:00, 00:00 по МСК)

## Темы статей

Список тем задан в `TOPICS` в `index.py`. Его можно менять под свои нужды.
