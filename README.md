# django-layout-development

Initial repository setup for pr-poehali-dev/django-layout-development

## Стек

- **Next.js 14** (App Router) — серверный рендеринг и маршрутизация
- React 18, TypeScript, Tailwind CSS, shadcn/ui

## Разработка и сборка

- `npm run dev` — запуск dev-сервера Next.js (http://localhost:3000)
- `npm run build` — сборка и генерация sitemap в `public/sitemap.xml`
- `npm run start` — запуск production-сервера после `build`
