# Почему контент не публикуется — схемы БД

## Решение (актуально)

Все функции (gallery, auto-blog, sitemap) **жёстко используют схему `public`** в SQL. Переменная `DB_SCHEMA` не используется.

Таблицы должны находиться в схеме `public`. Если миграции создали таблицы в другой схеме — перенеси их или пересоздай в `public`.

## Проверка

Вызови auto-blog вручную. Статья должна появиться на /blog и в админке.
