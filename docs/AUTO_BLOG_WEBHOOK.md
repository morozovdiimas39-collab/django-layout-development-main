# Публикация статей через админку (webhook)

В админке на вкладке «Блог» есть кнопка **«Сгенерировать (Gemini)»**. Она вызывает auto-blog через gallery.

## Настройка

1. В Yandex Cloud у функции **auto-blog** должен быть HTTP-триггер. Скопируй URL триггера.

2. У функции **gallery** в переменных окружения добавь:
   - `AUTO_BLOG_URL` = URL HTTP-триггера auto-blog (например `https://functions.yandexcloud.net/d4e...`)

3. Задеплой gallery после добавления переменной.

После этого кнопка «Сгенерировать (Gemini)» в админке будет вызывать auto-blog, статья появится в blog_posts и отобразится на /blog и в админке.
